ghkeys 
======

ghkeys is a tiny utility to get the public ssh keys of a github user.

Usage
-----
```bash
$ go get github.com/blang/ghkeys

// Get my ssh keys
$ ghkeys blang >> ~/.ssh/authorized_keys
```
